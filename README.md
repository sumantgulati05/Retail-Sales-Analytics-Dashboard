# Retail Sales Analytics Dashboard

## Overview
Built a 3-page interactive Power BI dashboard using the Superstore dataset (9,994 transactions) to analyze sales, profitability, customer behavior and geographic performance.

## Tools Used
- Power BI
- DAX
- Excel

## Key KPIs
- Total Sales: $2.3M
- Total Profit: $286K
- Profit Margin: 12.47%
- Total Orders: 5,009
- Average Order Value: $459

## Dashboard Pages

### 1. Executive Overview
- Sales KPIs
- Profitability Tracking
- Sales Trends
- Regional Performance

### 2. Product & Customer Analysis
- Profit by Sub-Category
- Sales by Sub-Category
- Top Customers
- Discount Impact Analysis

### 3. Geographic Analysis
- State-Level Sales Distribution
- Regional Sales & Profit Analysis
- Top Cities by Revenue

## Business Insights
- West region generated the highest sales.
- Technology category delivered the highest profit.
- Tables and Bookcases showed negative profitability.
- Consumer segment contributed the largest share of revenue.
